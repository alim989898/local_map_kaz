# local_map_kaz
Kazakhstan Local MBTiles Map (PHP 7.2)  Локальная веб-карта Казахстана на базе MapLibre GL JS и MBTiles.  Проект предназначен для работы полностью на локальном сервере без подключения к внешним картографическим сервисам. Тайлы хранятся в формате MBTiles и отдаются через PHP-скрипт.

Для GitHub можно оформить так:

Kazakhstan Local MBTiles Map (PHP 7.2)

Локальная веб-карта Казахстана на базе MapLibre GL JS и MBTiles.

Проект предназначен для работы полностью на локальном сервере без подключения к внешним картографическим сервисам. Тайлы хранятся в формате MBTiles и отдаются через PHP-скрипт.

Запуск
Скопировать проект на локальный веб-сервер.
Поместить файл kazakhstan.mbtiles.
Настроить путь к шрифтам.
Открыть:
http://localhost/

или

http://ваш_сервер/
Особенности

Данный проект является картографической подложкой и предназначен как основа для дальнейшей разработки:

аналогов 2GIS;
корпоративных ГИС;
систем мониторинга транспорта;
отображения объектов недвижимости;
геоинформационных веб-приложений.
Лицензия

Данные карты основаны на данных OpenStreetMap (ODbL).
