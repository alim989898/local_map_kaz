# Kazakhstan Local MBTiles Map

Локальная карта Казахстана на базе MBTiles, SQLite, PHP 7.2 и MapLibre GL JS.

Проект позволяет отображать карту Казахстана полностью локально без использования внешних картографических сервисов. Векторные тайлы хранятся в MBTiles и выдаются через PHP.

## Возможности

* Полностью офлайн работа
* Векторные MBTiles тайлы
* Отображение дорог
* Отображение зданий
* Отображение водоёмов
* Отображение населённых пунктов
* Просмотр информации об объектах по клику
* Поддержка русских названий
* Быстрая работа благодаря SQLite
* Совместимость с PHP 7.2

## Используемые технологии

* PHP 7.2
* SQLite3
* MBTiles
* MapLibre GL JS
* OpenStreetMap

## Требования

* PHP 7.2+
* SQLite3 Extension
* Apache/Nginx
* Современный браузер

## Запуск

1. Скопируйте проект на локальный сервер.
2. Поместите файл `kazakhstan.mbtiles` в каталог проекта.
3. Настройте путь к шрифтам.
4. Запустите веб-сервер.
5. Откройте браузер:

```text
http://localhost/
```

или

```text
http://your-server/
```
